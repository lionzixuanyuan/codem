What is it?

Codem is an open source project, designed to convert video material from one format to another, as flexible and as scalable as possible.

How does it work?

Codem consists of two separate projects, the Scheduler and the Transcoder. A Scheduler handles the scheduling and processing of jobs, and the Transcoder actually transcodes videos from one format to another.

The Scheduler

The Scheduler is built in Ruby on Rails and consists of a frontend viewable in your browser, and an API to handle scheduling of jobs via the command line. Read more about the Scheduler.

The Transcoder

The Transcoder is a NodeJS server, handling the actual transcoding of jobs through ffmpeg. Read more about the Transcoder.
