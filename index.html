What is it?

Codem is a free open source suite for converting video material from one format to another, as flexible and as scalable as possible.

How does it work?

Codem consists of two separate projects, the Scheduler and the Transcoder. The Scheduler handles the scheduling and handling of jobs, and the Transcoder transcodes videos from one format to another. These two components communicate via a lightweight JSON api over HTTP.

Why should I use it?

These are the key features of Codem:

  1. it's easy to add new video formats
  2. it's easy to scale out by adding more Transcoders and/or Schedulers
  3. it's easy to extend
