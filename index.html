What is it?

Codem is a free open source suite for converting video material from one format to another, as flexible and as scalable as possible.

How does it work?

Codem consists of two separate projects, the Scheduler and the Transcoder. The Scheduler handles the scheduling and handling of jobs, and the Transcoder transcodes videos from one format to another. These two components communicate via a lightweight JSON api over HTTP. It's possible to use these two packages independently, but they are designed to work together. Read more about the architecture of Codem here.

Why should I use it?

Codem is designed to be a flexible, scalable and robust solution to handle video transcoding.
These are the key features of Codem:

  1. it's simple to add new video formats
  2. it's simple to add more processing power
  3. it's simple to extend
  4. it's designed to be robust
  5. it's hosted on your own hardware
