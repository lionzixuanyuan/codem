Hosts

Overview

A host in the Scheduler represents a Transcoder server running somewhere. Hosts can be added through the API and the web front end.

To create a host via the API, use the following command:

{% highlight bash %} $ curl -d 'name=transcoder&url=http://transcoder.com' http://localhost:3000/api/hosts {% endhighlight %}

The Scheduler will respond with something like:

{% highlight bash %} { "host": { "available":true, "available_slots":10, "created_at":"2011-05-09T15:32:53Z", "id":6, "name":"transcoder", "total_slots":10, "updated_at":"2011-05-09T15:32:53Z", "url":"http://transcoder.com" } } {% endhighlight %}

For more information, see the API documentation of the Scheduler here.
