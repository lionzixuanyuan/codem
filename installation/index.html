--- title: Installation of Codem layout: default ---

Installing Codem is a three step process.

  1. Install the prerequisites
  2. Install the Transcoder
  3. Install the Scheduler

1. Prerequisites

Packages listed here are required to run the Codem software suite.

  1. ruby
  2. node js
  3. npm
  4. ffmpeg
  5. sqlite

git is required if you want to download and work on the source code of Codem.

2. Installing the Transcoder

  1. Either clone the git repository or download a packaged archive. {% highlight bash %} $ git clone git://github.com/NPO/codem-transcode.git {% endhighlight %}
  2. Install dependencies through npm {% highlight bash %} $ npm install sqlite3 connect argsparser mkdirp {% endhighlight %}
  3. Start the server {% highlight bash %} $ node transcoder.js {% endhighlight %}

For more information, see the Transcoder README

3. Installing the Scheduler

  1. Either clone the git repository or download a packaged archive. {% highlight bash %} $ git clone git://github.com/NPO/codem-schedule.git {% endhighlight %}
  2. Install the required gems. {% highlight bash %} $ bundle install {% endhighlight %}
  3. Run the installer rake task, which will setup the application. {% highlight bash %} $ rake codem:install {% endhighlight %}
  4. Use cron or a similar tool to trigger the status update task. For example, to run the task every 2 minutes, use: {% highlight bash %} */2 * * * * curl -s http://localhost:3000/api/schedule {% endhighlight %}
  5. Start the server {% highlight bash %} $ rails server {% endhighlight %}

For more information, see the Scheduler README
